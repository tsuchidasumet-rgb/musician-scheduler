# 🎸 Musician Scheduler

An intelligent scheduling solution built with **Google Apps Script** and **LINE Messaging API**, designed to streamline shift management for professional musicians.

## ✨ Key Features
- **Smart Auto-Draft:** Round-robin scheduling logic that considers musician roles and availability.
- **Automatic Leave Calculation:** Automatically marks non-working days as "Leave" to ensure schedule integrity.
- **LINE Bot Integration:** Musicians can report leave or lock dates directly via LINE chat.
- **Multi-view Web Dashboard:**
  - **Admin View:** Full schedule control and auto-generation.
  - **Member View:** Shift reporting and registration.
  - **Personal View:** Individual schedule checking.

## 🛠 Tech Stack
- **Backend:** Google Apps Script (JavaScript V8)
- **Database:** Google Sheets
- **Frontend:** HTML5, CSS3, Bootstrap 5, Flatpickr
- **Integration:** LINE Messaging API
